#### Introduction - 简介

本书的主要内容是大学生计算机算法，借鉴了一些经典书籍和资料。通过公式、插图来描述算法，配合源码、测试、OJ题目，清晰易懂。
由于自己经验有限，难免存在疏漏和错误，还请指正。读者的支持是本项目更新的动力，欢迎大家给星、分享以及参与编写。
